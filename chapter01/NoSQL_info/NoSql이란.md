# 1. NoSql이란?

---
- Not Only Sql의 약자로 관계형 데이터베이스에서의 SQL을 사용하지 않는 데이터 저장소
- 여러 타입의 NoSQL의 공통적인 특징 -> 관계가 정의돼 있지 않은 데이터를 저장한다.
- 마이크로서비스의 저장소로 쓰일 수 있도록 여러 요구 사항을 만족
